# Knowledge Maturity Model

## Tujuan

Menetapkan lima tingkat kematangan knowledge.

## Levels

| Level | Name | Description |
|---|---|---|
| 1 | Documented | Dokumen tersedia |
| 2 | Structured | Metadata konsisten |
| 3 | Connected | Relasi lintas domain |
| 4 | Intelligent | AI dapat menalar |
| 5 | Compounding | Penggunaan menghasilkan pembelajaran baru |
