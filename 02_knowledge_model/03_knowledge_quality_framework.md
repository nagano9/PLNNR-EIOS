# Knowledge Quality Framework

## Tujuan

Menetapkan dimensi kualitas pengetahuan.

## Dimensi

- Accuracy
- Completeness
- Timeliness
- Consistency
- Traceability
- Relevance
- Authority
- Reusability

## Scoring

Setiap dimensi dinilai 1–5. Knowledge object material harus mencapai skor minimum yang ditetapkan domain owner.

## Ambang Minimum (disarankan)

| Konteks penggunaan | Ambang minimum rata-rata |
|---|---|
| Keputusan investasi/keuangan material | ≥ 4.0 dan Authority ≥ 4 |
| Analisis pendukung | ≥ 3.5 |
| Referensi umum internal | ≥ 3.0 |

Dimensi **Authority** ditentukan oleh `04_source_authority_hierarchy.md`. Object di bawah ambang tidak boleh menjadi dasar tunggal keputusan material.
