# Core Entity Definitions

## Tujuan

Memberikan definisi awal entitas utama.

## Definitions

| Entity | Definition |
|---|---|
| Project | Inisiatif terstruktur dengan outcome, biaya, waktu, dan risiko |
| Asset | Sumber daya ekonomi yang dikendalikan |
| Investment | Penempatan modal untuk menghasilkan nilai |
| Contract | Perikatan yang menciptakan hak dan kewajiban |
| Decision | Pilihan yang disahkan oleh otoritas |
| Assumption | Kondisi yang diterima sebagai dasar analisis |
| Risk | Ketidakpastian yang memengaruhi tujuan |
| Lesson | Pengetahuan dari pengalaman aktual |

## Entitas Khas Investment Holding (PLN NR)

| Entity | Definition |
|---|---|
| JVC | Joint Venture Company tempat PLN NR menjadi pemegang saham (mayoritas/minoritas) |
| Shareholding | Kepemilikan saham PLN NR pada JVC (persentase, kelas, hak) |
| Representative | Wakil PLN NR di Direksi/Komisaris JVC |
| GovernanceGate | Titik keputusan: DMR, Direksi, Dekom, RUPS, RUPS JVC |
| AssignmentType | Klasifikasi Penugasan vs Non-Penugasan |
| ReservedMatter | Hal yang memerlukan persetujuan khusus/proteksi pemegang saham |
| RegulatoryWatchItem | Regulasi yang dipantau berkala (tarif EBT, threshold aksi korporasi, POJK) |
