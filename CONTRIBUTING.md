# Contribution Standard

## Prinsip
Setiap kontribusi harus:
- memiliki tujuan bisnis yang jelas;
- menggunakan metadata minimum;
- menyebutkan sumber;
- menyatakan status validasi;
- memiliki owner;
- menggunakan ID unik;
- menghindari duplikasi;
- menjaga klasifikasi informasi.

## Workflow
1. Buat branch kerja.
2. Gunakan template yang sesuai.
3. Jalankan quality checklist.
4. Ajukan pull request.
5. Minta review domain owner dan knowledge steward.
6. Merge setelah seluruh mandatory check terpenuhi.

## Definition of Done
Artefak dianggap selesai apabila:
- isi substantif tersedia;
- metadata lengkap;
- relasi utama tercatat;
- sumber dapat ditelusuri;
- status validasi jelas;
- tidak ada informasi rahasia pada repositori publik.
