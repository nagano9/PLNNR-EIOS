# Naming Convention

## Tujuan

Menetapkan standar nama file, folder, dan object.

## File Naming

`<domain>_<object-type>_<short-title>_<version>.<ext>`

## Object ID

`<DOMAIN>-<TYPE>-<YYYY>-<NNNN>`
Contoh: `FIN-METRIC-2026-0001`.

## Rules

Gunakan huruf kecil untuk nama file, underscore sebagai pemisah, dan hindari karakter khusus.

## Kode Domain (DOMAIN)

`GOV` governance, `ARC` architecture, `FIN` finance, `INV` investment, `PROJ` project, `COM` commercial, `LEG` legal, `RISK` risk, `ASSET` asset, `SUS` sustainability, `TECH` technology, `AI` ai, `DEC` decision, `KM` knowledge.

## Contoh PLN NR

- `PROJ-FACT-2026-0012` — fakta proyek PLTS Terapung Cirata.
- `DEC-INV-2026-0001` — keputusan struktur investasi.
- `FIN-METRIC-2026-0001` — metrik DSCR proyek.

Object ID bersifat immutable setelah dipublikasikan.
