# Enterprise Metadata Standard

## Tujuan

Menetapkan metadata minimum untuk seluruh artefak EIOS.

## Mandatory Fields

| Field | Description |
|---|---|
| object_id | ID unik |
| title | Nama artefak |
| object_type | Jenis object |
| domain | Domain bisnis |
| owner | Pemilik substansi |
| steward | Penjaga kualitas |
| source | Sumber primer |
| status | Lifecycle status |
| confidentiality | Klasifikasi |
| validation_status | Status validasi |
| effective_date | Tanggal berlaku |
| version | Versi |

## Optional Fields

Project, entity, year, scenario, jurisdiction, counterparty, currency, unit, related objects.
